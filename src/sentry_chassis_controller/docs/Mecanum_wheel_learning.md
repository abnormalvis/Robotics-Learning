# 麦克纳姆轮（Mecanum Wheel）

**麦克纳姆轮**是一种**全向轮**设计，可使陆地载具在平面内**任意方向移动**。又称**瑞典轮**或**伊隆轮**，以其发明者、瑞典 Mecanum AB 公司工程师 Bengt Erland Ilon（1923–2008）命名。他于 1972 年 11 月 13 日在美国获得专利。

---

## 1. 结构原理

- 轮胎外围均匀布置一系列**橡胶辊子**，辊子轴线与轮毂平面成 **45°**，也与车轴成 **45°**。
- 每个车轮由**独立电机**驱动，无需转向机构。
- 通过四个车轮不同转速/转向的组合，将各辊子产生的**斜向推力**合成为车体的**任意平动+旋转**。

标准布置：  
**四轮方案**，左前、右后装“左手轮”；右前、左后装“右手轮”。  
（所谓“手性”指辊子倾斜方向）

---

## 2. 典型应用

| 场景 | 说明 |
|---|---|
| **叉车** | 超窄巷道、原地调头 |
| **自主移动机器人（AMR）** | 竞赛、仓储、巡检 |
| **轮椅** | 2006 EVER Monaco 展示过全向轮椅 |
| **航空容器装卸车** | 美海军航母甲板、民航货站 |
| **青少年机器人赛事** | FTC、VEX 等队伍广泛采用 |

---

## 3. 历史里程碑

| 时间 | 事件 |
|---|---|
| 1972 | Ilon 提交美国专利（US 3,876,255） |
| 1980s | 美海军买下专利，巴拿马城基地继续研发 |
| 1997 | Airtrax 等公司支付 2 500 美元获得民用化授权 |
| 1985 | 卡内基梅隆大学 CMU URANUS 机器人首次安装麦克纳姆轮，用于导航研究 |
| 1992 | CMU“Tessellator”航天飞机隔热瓦检修机器人采用该轮 |
| 2006 | EVER Monaco 展出配备麦克纳姆轮的电动轮椅 |

---

## 4. 优势对比

| 项目 | 麦克纳姆轮 | 履带/滑移转向 |
|---|---|---|
| **转向方式** | 全向矢量合成 | 打滑差速 |
| **地面划伤** | 极小 | 可能严重 |
| **所需扭矩** | 低 | 高（需克服滑动摩擦） |
| **结构复杂度** | 四轮独立电机 | 简单但需履带系统 |
